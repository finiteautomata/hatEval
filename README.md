# hatEval
Atalaya at hatEval

# Instructions

1. Get data

```
./bin/get_data.sh
```

2. Get Elmo

```
./bin/get_elmo.sh
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Install this package in editable mode

```
pip install -e .
```

5. Run tests
```
python setup.py test
```
